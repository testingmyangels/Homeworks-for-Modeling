This Python program demonstrates a basic conditional statement (if-else) to classify a person as an adult or a minor based on their age.
The program assigns a value to the variable age and then evaluates whether it is greater than or equal to 18. If the condition is satisfied, it prints that the person is an adult. Otherwise, it prints that the person is a minor.
This example introduces fundamental programming concepts such as variables, comparison operators, and control flow, making it suitable for beginners learning Python.
